Blogr – Landing Page 

Esse projeto é uma landing page responsiva, desenvolvida como parte de um desafio do Frontend Mentor.

A ideia aqui foi treinar HTML, CSS e JavaScript puro, focando principalmente em layout, responsividade e interações de interface.

Por que eu fiz esse projeto?

Quis praticar:

Organização de CSS em arquivos separados

Layouts modernos usando Flexbox e Grid

Menus dropdown no desktop e mobile

Responsividade real (não só “quebrar linha”)

JavaScript simples para controlar estado de UI

Além disso, o design do Blogr é ótimo pra treinar atenção aos detalhes.

O que você vai encontrar aqui

Hero section com call-to-action

Menu com dropdown animado (desktop e mobile)

Seções alternando texto e ilustrações

Layout totalmente responsivo

Footer organizado e fácil de navegar

Tudo feito com código limpo e sem dependências externas.

Tecnologias usadas

HTML5 (estrutura semântica)

CSS3

Flexbox

Grid Layout

Media Queries

Variáveis CSS

Animações

JavaScript (Vanilla)

Google Fonts

Responsividade
